## 题目描述

有一个 $N$ 行 $N$ 列的网格，每个方格要么是白色要么是黑色。

网格的状态由 $N$ 个字符串 $S_i$ 表示。如果 $S_i$ 的第 $j$ 个字符是 `#`，则从顶部数第 $i$ 行、从左边数第 $j$ 列的方格被涂成黑色。如果字符是 `.`，则方格被涂成白色。

高桥可以选择至多两个白色方格，将它们涂成黑色。

判断是否可能使得网格中包含至少 6 个连续的黑色方格，无论是水平、垂直还是对角线排列。

这里，如果具有 $N$ 行 $N$ 列的网格完全包含具有 6 行 6 列的子网格，使得其对角线上的所有方格都被涂成黑色，则称网格中包含至少 6 个连续的黑色方格。

## 限制条件

- $6 \le N \le 1000$
- $|S_i| = N$
- $S_i$ 由 `#` 和 `.` 组成。

## 输入

输入以以下格式从标准输入给出：

```text
N
S_1
S_2
:
S_N
```

## 输出
如果通过涂至多两个方格可以满足条件，则输出`Yes`,否则输出`N`。

## 解题思路
* 因为是1000 * 1000 的网格，所以可以直接暴力，但是本题主要是为了优化暴力的代码操作的.
* 正常来说的话，就是横向判断一个for, 纵向一个for，两个斜向分别是两个for
* 但是实际上我们可以对每个点直接跑四个dir， 每个点往下，往右，往左下，往右下跑四个方向，每个方向跑六个。所以复杂度是  $O (6N^2)$, 也就是`1000 * 1000 * 6`. 完全够用了.

## 代码中间值得注意的是！！
* 首先`vector`不要开成int了，int就直接wa了，这个bug还非常的阴间，一般不会有人去注意到这个东西的
* 第二个bug就是，在check的时候，当你找到了`cnt >= 4`的时候，不要很着急`return true`回来，因为这连续四个可能是卡在边界的。所以我们要在边界判断那里再加个筹码，
如果边界超了的话，`found`直接就变成`false`.这样即便`cnt >= 4`的时候被置`true`了。这个判断也能及时的改回`false`.


## 代码如下:
```cpp
#include <bits/stdc++.h>
using namespace std;

int n;
const int MAXN = 1005;

int dx[] = {1, 0, 1, 1};
int dy[] = {0, 1, -1, 1};
 
vector <vector <char>> a (MAXN + 1, vector <char> (MAXN + 1));

bool check (int x, int y) {
    for (int i = 0; i < 4; i ++) {
        int cnt = 0; // 计算#
        bool found = false;
        for (int step = 0; step < 6; step ++) {
            int cx = x + dx[i] * step;
            int cy = y + dy[i] * step;

            if (cx < 1 || cx > n || cy < 1 || cy > n) {
                found = false;
                break;
            } 
            if (a[cx][cy] == '#') {
                cnt ++;
            }
            if (cnt >= 4) {
                found = true;
            }
        }
        if (found)  return true;
    }
    return false;
}

int main ()
{
    ios::sync_with_stdio (false);
    cin.tie (0);

    cin >> n;

    for (int i = 1; i <= n; i ++) {
        for (int j = 1; j <= n; j ++) {
            cin >> a[i][j];
        }
    }
    bool found = false;
    for (int i = 1; i <= n; i ++) {
        for (int j = 1; j <= n; j ++) {
            if (check (i, j) == true) {
                cout << "Yes" << '\n';
                return 0;
            }
        }
    }
    cout << "No" << '\n';
    return 0;
}
```
