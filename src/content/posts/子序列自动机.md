### 题目描述

给你两个由小写英文字母组成的字符串 $S$ 和 $T$。

请统计 $S$ 的所有非空子串 $s$ 中，有多少个**不包含** $T$ 作为（不一定连续的）子序列。

> **注意：** $S$ 的两个子串如果取自不同的位置，即使字符串内容相同，也算作不同的子串。

### 约束条件

- $S$ 是由小写英文字母组成的字符串。
- $1 \le |S| \le 2 \times 10^5$，其中 $|S|$ 是 $S$ 的长度。
- $T$ 是由小写英文字母组成的字符串。
- $1 \le |T| \le 50$，其中 $|T|$ 是 $T$ 的长度。



---
### 输入格式

输入从标准输入给出，格式如下：
```text
S
T
```

### 输出格式
输出：
```text
51
```


### 题解思路

首先有两个坑点要注意一下的：

1. **字串还是子序列？**：“子串 (必须连续)”      “子序列 (可不连续)。
2. **暴力是不可行的**：试图枚举 $S$ 的所有子串，然后依次判断是否包含 $T$，这会导致 $O(|S|^2 \times |T|)$ 的时间复杂度，对于 $|S| = 2 \times 10^5$ 的数据量绝对会超时。

那这个题目我们到底要怎么去思考呢：

#### 1. 利用单调性寻找“边界”
如果说现在我们找到某一个字串$K$，然后它恰好包含了我们的目标串$T$, 我们把这个$K$的左边界叫做$L$, 右边界叫做$R$.

那是不是也就意味着,如果我把$R$再往右取,后面的所有的子串都是包含目标串$T$的.

也就是说：如果从 $L$ 到某个终点位置（假设为 $R_{min}$）**刚刚好**完整凑出了子序列 $T$，那么：
* $R_{min}$ 以及它右边的所有位置，作为终点都是**不合法**的（因为都已经包含 $T$ 了）。
* 在 $R_{min}$ 左边的所有位置（即从 $L$ 到 $R_{min} - 1$），作为终点都是**合法**的（因为还没凑齐 $T$）。



题目要求计算“不包含 $T$ 的子串数量”。既然子串必须是连续的，我们不妨**固定子串的起点 $L$**，寻找它合法的终点 $R$, 然后我们把每个起点$L$开始的合法子串数量累加到$ans$里面去就好了。

因此，对于每一个起点 $L$，我们只需要找出**最早能凑齐 $T$ 的那个终点位置 $R_{min}$**，那么以 $L$ 为起点的合法子串数量就是：
$$合法的终点个数 = (R_{min} - 1) - L + 1 = R_{min} - L$$

---
#### 2. 子序列自动机加速查找
现在的问题变成了：**如何快速找到每个起点 $L$ 对应的 $R_{min}$？**

如果每次都从 $L$ 开始往右遍历去寻找 $T$ 的字符，最坏情况下时间复杂度依然会退化。为了实现 $O(1)$ 的查找，我们需要利用空间换时间，引入**子序列自动机（nxt 数组）**进行预处理。

我们定义 `nxt[i][c]` 表示：在字符串 $S$ 的第 $i$ 个位置（含）及之后，字符 $c$ **第一次出现的位置索引**。
为了快速构建这个数组，我们采用**倒序遍历**的思想：
* 从字符串末尾往前遍历。
* 当前位置的 `nxt` 状态，可以直接“抄作业”继承自 $i+1$ 位置的状态。
* 然后单独把当前字符 $S[i]$ 的出现位置更新为 $i$ 即可。

我们可以举个例子来理解一下这个操作逻辑,假如我们现在有一个串absdabbaad,$len = 10$,那现在看到倒数第三个char,也就是a,如果我要查找nex[7][a],那我
可以执行的操作就是,令`nex[7][a]` = `nex[8][a]`, 也就是继承后面位置的状态,然后再更新自己的状态, 令`nex[7][S[7]] = 7`,(这个操作很像动态规划)

有了 `nxt` 数组，当我们从起点 $L$ 开始匹配 $T$ 时，就像青蛙跳荷叶一样，可以直接跳到下一个需要的字符所在的位置，最多跳 $|T|$ 次就能找到 $R_{min}$，极其高效。


### 核心代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio (false);
    cin.tie(0);

    string s, t;
    cin >> s >> t;

    int n = s.length();
    int m = t.length();

    vector<vector<int>> nxt(n + 1, vector<int>(26, n));

    for (int i = n - 1; i >= 0; i --) {
        for (int c = 0; c < 26; c++) {
            nxt[i][c] = nxt[i + 1][c];
        }
        // 更新一下自己位置的值
        nxt[i][s[i] - 'a'] = i;
    }
    

    long long ans = 0;
    for (int l = 0; l < n; l ++) {
        int curr = l;
        bool found = true;

        for (int i = 0; i < m; i ++) {
            if (curr >= n || nxt[curr][t[i] - 'a'] == n) {
                found = false;
                break;
            }
            curr = nxt[curr][t[i] - 'a'] + 1;
        }
        if (found) {
            // 以l为开头的合法字串的总数.
            ans += (curr - 1 - l);
        } else {
            ans += (n - l);
        }
    }
    cout << ans << '\n';

    return 0;
}
```
