### 题目描述

给你两个由小写英文字母组成的字符串 $S$ 和 $T$。

请统计 $S$ 的所有非空子串 $s$ 中，有多少个**不包含** $T$ 作为（不一定连续的）子序列。

> **注意：** $S$ 的两个子串如果取自不同的位置，即使字符串内容相同，也算作不同的子串。

### 约束条件

- $S$ 是由小写英文字母组成的字符串。
- $1 \le |S| \le 2 \times 10^5$，其中 $|S|$ 是 $S$ 的长度。
- $T$ 是由小写英文字母组成的字符串。
- $1 \le |T| \le 50$，其中 $|T|$ 是 $T$ 的长度。



---
### 输入格式

输入从标准输入给出，格式如下：
```text
S
T
```

### 输出格式
输出：
```text
51
```


### 题解思路




### 核心代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio (false);
    cin.tie(0);

    string s, t;
    cin >> s >> t;

    int n = s.length();
    int m = t.length();

    vector<vector<int>> nxt(n + 1, vector<int>(26, n));

    for (int i = n - 1; i >= 0; i --) {
        for (int c = 0; c < 26; c++) {
            nxt[i][c] = nxt[i + 1][c];
        }
        // 更新一下自己位置的值
        nxt[i][s[i] - 'a'] = i;
    }
    

    long long ans = 0;
    for (int l = 0; l < n; l ++) {
        int curr = l;
        bool found = true;

        for (int i = 0; i < m; i ++) {
            if (curr >= n || nxt[curr][t[i] - 'a'] == n) {
                found = false;
                break;
            }
            curr = nxt[curr][t[i] - 'a'] + 1;
        }
        if (found) {
            // 以l为开头的合法字串的总数.
            ans += (curr - 1 - l);
        } else {
            ans += (n - l);
        }
    }
    cout << ans << '\n';

    return 0;
}
```
