## 1. 算法概述

树上背包（Tree Knapsack）是树形 DP 与背包 DP 的结合体。
**典型场景**：给定一棵树，每个节点有重量（代价）和价值，要求在满足总重量不超过 $V$ 的前提下，且**选择某个节点必须先选择其父节点**（或类似依赖关系），求能获得的最大价值。

**核心思想**：在 DFS 的回溯阶段（后序遍历），将每个子树视为一个“物品组”，在当前节点所在的作用域内跑一遍**分组 01 背包**。

## 题目

小明正在改造一个生产车间的生产流水线。这个车间共有 $n$ 台设备，构成以 $1$ 为根节点的一棵树，结点 $i$ 有权值 $w_i$。其中叶节点的权值 $w_i$ 表示每单位时间将产出 $w_i$ 单位的材料并送往父结点，根结点的权值 $w_i$ 表示每单位时间内能打包多少单位成品，其他结点的权值 $w_i$ 表示每单位时间最多能加工 $w_i$ 单位的材料并送往父结点。

由于当前生产线中某些结点存在产能不够的问题导致生产线无法正常运行，即存在某些结点每单位时间收到的材料超过了当前结点的加工能力上限。小明计划删除一些结点使得所有结点都能正常运行。他想知道删除一些结点后根结点每单位时间内最多能打包多少单位的成品？

## Input

输入共 $n + 1$ 行。

第一行为一个正整数 $n$。

第二行为 $n$ 个由空格分开的正整数 $w_1, w_2, \ldots, w_n$。

后面 $n - 1$ 行，每行两个整数表示树上的一条边连接的两个结点。

## Output

输出共一行，一个整数代表答案。

## 2. 状态定义与转移方程

这是一棵树，而且父节点的状态完全取决于子节点传上来的状态，这显然提示我们要用 **树形 DP（自底向上 DFS）**。

### 1. 状态定义

对于节点 `u`，它的子树可能有很多种合法的组合方式，从而向 `u` 提供不同的产量。我们需要记录所有可能的产量值。

*   定义 `dp[u][j]`：表示以 `u` 为根的子树，是否能够向它的父节点提供恰好为 `j` 的产量。其值为 `true` 或 `false`。

### 2. 边界条件（叶子节点）

叶子节点没有子节点，它的产量由自己决定。

*   它可以正常工作，提供 $w_u$ 的产量：`dp[u][w_u] = true`
*   它可以被删除，提供 0 的产量：`dp[u][0] = true`

### 3. 状态转移（核心：树上背包）

对于非叶子节点 `u`，它需要把所有子节点 `v` 的产量加起来。
这本质上是一个背包问题：

*   **背包容量**：父节点 `u` 的容量限制 $w_u$。
*   **物品**：`u` 的每一个子节点 `v`。
*   **物品的重量**：子节点 `v` 能提供的各种可能的产量 `j`（即满足 `dp[v][j] == true` 的 `j`）。

我们要从每个子节点中挑出一个合法的产量值，把它们加起来。如果累加和 $\le w_u$，那么这个累加和就是父节点 `u` 可以向更上层提供的一个合法产量。

### 4. “删除节点”的兜底机制

题意说可以删除任意节点。如果父节点 `u` 发现无论怎么组合子节点，总产量都超标了，或者即使没超标，我们依然可以选择直接把节点 `u` 删掉。

*   这意味着对于任何节点 `u`，无论其子节点如何，`dp[u][0] = true` 永远成立。

## 3. C++ 标准模板 (通用最大价值版)

下面这份模板采用了“新旧数组交替”的写法，逻辑最清晰，最不容易在体积的遍历顺序上翻车。

```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 1005;
vector <int> adj [MAXN];
vector <int> w (MAXN + 1);
bool dp[MAXN][MAXN];

void dfs (int cur, int fa) {
    if (adj[cur].size () == 1 && cur != 1) {
        dp[cur][0] =  true;
        dp[cur][w[cur]] = true;
        // 叶子节点可以提供自己全部 ,或者一个都提供不了
        return;
    }

    // 记录cur节点可以提供的所有点数
    vector <bool> M_book (w[cur] + 1, false);
    M_book[0] = true;
    
    for (auto it : adj[cur]) {
        if (it == fa) continue;

        dfs (it, cur);
        vector <bool> N_book (w[cur] + 1, false);// 每个子节点都拷贝一本book记录

        for (int i = 0; i <= w[cur]; i ++) {// 查看cur有哪些点是可用的
            if (!M_book[i]) continue;

            for (int j = 0; j <= w[it]; j ++) {// 查看孩子节点有哪些点是可用的
                if (!dp[it][j]) continue;

                // 到这里就说明父亲和孩子都可以用
                if (i + j <= w[cur]) {
                    N_book[i + j] = true;
                }
            }
        }
        // 到这就找完一个子节点了，那现在把新数据整合到M_book
        M_book = N_book;
    }

    // 到这里cur节点已经遍历完所有的子节点，所有可提供点数信息都在M_book
    // 现在把他们更新到dp里面

    for (int i = 0; i <= w[cur]; i ++) {
        if (M_book[i] != 0) {
            dp[cur][i] = true;
        }
    }
    dp[cur][0] = true;
}

int main ()
{
    ios::sync_with_stdio (false);
    cin.tie (0);

    int n;
    cin >> n;
    
    for (int i = 1; i <= n; i ++) {
        cin >> w[i];
    }
    for (int i = 1; i < n; i ++) {
        int u, v;
        cin >> u >> v;

        adj[u].push_back (v);
        adj[v].push_back (u);
    }
    
    dfs (1, 0);
    // 从根节点开始dfs,但是每次要先找到叶子，从叶子开始dp

    int ans = -1;
    for (int i = w[1]; i >= 0; i --) {
        if (dp[1][i]) {
            ans = i;
            break;
        }
    }
    cout << ans << '\n';
    return 0;
}
```
