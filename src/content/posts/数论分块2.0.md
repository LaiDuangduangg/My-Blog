## 题目描述

事后刺客成了“无名户”。晏政的姐姐听闻消息赶来验看，凭旧痕识出是弟，当众说明其为报严仲子之恩而死。说完便自尽，以明其言不虚，也全手足之义。

后来城里要把这件事写进史册，可偏偏“无名户”三个字最难写：写轻了怕埋没义士，写重了又怕惹祸。于是史官想了个法子：不在纸上直写人名，只用“数字暗号”来记。

他们把严仲子之恩记作一个整数 $n$。又把要写的条目按顺序编号为 $1,2,3,\dots$。每写到第 $i$ 条，就先算 $\left\lfloor \frac{n}{i} \right\rfloor$，再把这个数除以 $i$ 的余数记下来，也就是：

$$
\left(\left\lfloor \frac{n}{i} \right\rfloor \bmod i\right)
$$

若要抄录一段条目，从第 $l$ 条到第 $r$ 条，就把这一段里每条对应的余数加起来，定义：

$$
f(l,r)=\sum_{i=l}^{r}
\left(
\left\lfloor \frac{n}{i} \right\rfloor \bmod i
\right)
$$

现在给你 $q$ 次询问，每次询问给你两个整数 $l,r$，问你 $f(l,r)$ 的值对 $10^9+7$ 取模的结果是多少？

注：$\lfloor x \rfloor$ 表示不超过 $x$ 的最大整数。

## 输入格式

第一行两个整数 $n,q$：

$$
1 \le n \le 10^{12},\quad 1 \le q \le 3 \times 10^5
$$

接下来 $q$ 行，每行两个整数 $l,r$：

$$
1 \le l \le r \le n
$$

表示一组询问。

## 输出格式

输出 $q$ 行，每行一个整数，表示每组询问的答案。

## 样例 1

### 输入

```text
10 3
1 4
2 5
3 6
```

### 输出

```text
3
5
5
```

## 样例 2

### 输入

```text
1000000000000 1
3 8787899898
```

### 输出

```text
126362636
```

## 备注

对于样例 1 第一个询问：

$$
\left\lfloor \frac{10}{1} \right\rfloor \bmod 1
+
\left\lfloor \frac{10}{2} \right\rfloor \bmod 2
+
\left\lfloor \frac{10}{3} \right\rfloor \bmod 3
+
\left\lfloor \frac{10}{4} \right\rfloor \bmod 4
=
0+1+0+2=3
$$


## 题意简述

题目要求多次询问：

$$
f(l,r)=\sum_{i=l}^{r}
\left(
\left\lfloor \frac{n}{i} \right\rfloor \bmod i
\right)
$$

每次给出一组 $l,r$，需要求出 $f(l,r)$ 对 $10^9+7$ 取模后的值。

其中：

$$
1 \le n \le 10^{12}, \qquad 1 \le q \le 3 \times 10^5
$$

所以不能对每个询问都从 $l$ 枚举到 $r$。

## 1. 先把 `mod` 拆开

这道题最关键的第一步，是把：

$$
\left\lfloor \frac{n}{i} \right\rfloor \bmod i
$$

拆开。

根据取模的定义：

$$
a \bmod b = a - b \times \left\lfloor \frac{a}{b} \right\rfloor
$$

这里令：

$$
a = \left\lfloor \frac{n}{i} \right\rfloor,\qquad b=i
$$

那么：

$$
\left\lfloor \frac{n}{i} \right\rfloor \bmod i
=
\left\lfloor \frac{n}{i} \right\rfloor
-
i \times
\left\lfloor
\frac{\left\lfloor \frac{n}{i} \right\rfloor}{i}
\right\rfloor
$$

又因为：

$$
\left\lfloor
\frac{\left\lfloor \frac{n}{i} \right\rfloor}{i}
\right\rfloor
=
\left\lfloor \frac{n}{i^2} \right\rfloor
$$

所以原式可以化成：

$$
\left\lfloor \frac{n}{i} \right\rfloor \bmod i
=
\left\lfloor \frac{n}{i} \right\rfloor
-
i \times \left\lfloor \frac{n}{i^2} \right\rfloor
$$

于是：

$$
f(l,r)
=
\sum_{i=l}^{r}
\left\lfloor \frac{n}{i} \right\rfloor
-
\sum_{i=l}^{r}
i \times \left\lfloor \frac{n}{i^2} \right\rfloor
$$

也就是说，最后答案可以拆成两部分：

1. $\displaystyle \sum_{i=l}^{r}\left\lfloor \frac{n}{i} \right\rfloor$
2. $\displaystyle \sum_{i=l}^{r} i \times \left\lfloor \frac{n}{i^2} \right\rfloor$

最后用第一部分减去第二部分即可。

## 2. 第一部分：处理 $\sum \left\lfloor \frac{n}{i} \right\rfloor$

对于第一部分：

$$
\sum_{i=l}^{r}\left\lfloor \frac{n}{i} \right\rfloor
$$

很自然地想到用 **整除分块**。

因为在一段连续的 $i$ 中，$\left\lfloor \frac{n}{i} \right\rfloor$ 的值可能是相同的。

例如某些位置可以被划成：

```text
[a1, a2, a3, a4] [a5, a6, a7, ...] [a8, ...]
       val1              val2              val3
```

在同一块中，对应的值 `val` 是相同的。

假设当前块是 $[L,R]$，这一块的值是：

$$
val = \left\lfloor \frac{n}{L} \right\rfloor
$$

那么这一块里面的所有位置都有：

$$
\left\lfloor \frac{n}{i} \right\rfloor = val
$$

所以这一块的贡献就是：

$$
(R-L+1) \times val
$$

## 3. 预处理第一部分的前缀和

由于有很多次询问，我们不能每次询问都重新分块计算。

所以可以预处理每一个整除分块的：

```text
L, R, pre_sum
```

也就是对于每一块，我们记录：

- `L`：这一块的左端点；
- `R`：这一块的右端点；
- `pre_sum`：到当前块右端点为止，第一部分的前缀和。

比如可以想象成：
![数论分块示意图](/images/ba33b9bca9eb0f8429736ad81cb42f58.png)

注意：这里的 `pre` 是对每一块前面的所有贡献做前缀和。

这样对于每个询问的 $l$ 和 $r$，都可以用二分法快速定位它们分别在哪一块里面。

## 4. 查询第一部分时如何使用前缀和

假设我们已经预处理好了每一块的 `L`、`R`、`pre_sum`。

对于一个位置 $x$，如果要计算：

$$
\sum_{i=1}^{x}\left\lfloor \frac{n}{i} \right\rfloor
$$

就可以先二分找到 $x$ 所在的那一块。

设这一块是：

```text
[L, R]
```

这一块的值是：

```text
val
```

那么：

```text
1 ... L-1 | L ... x ... R
```

前面完整块的贡献可以直接用前缀和拿到。

当前块只需要加上从 $L$ 到 $x$ 的这一段：

$$
(x-L+1) \times val
$$

所以：

$$
sum_1(x)
=
pre_{\text{上一块}}
+
(x-L+1)\times val
$$

于是区间 $[l,r]$ 的第一部分就是：

$$
sum_1(r)-sum_1(l-1)
$$

## 5. 第二部分：处理 $\sum i \times \left\lfloor \frac{n}{i^2} \right\rfloor$

第二部分是：

$$
\sum_{i=l}^{r}
i \times \left\lfloor \frac{n}{i^2} \right\rfloor
$$

这里要注意：

当 $i^2 > n$ 时：

$$
\left\lfloor \frac{n}{i^2} \right\rfloor = 0
$$

也就是说，后面全是 $0$，没有意义。

所以这一部分只需要处理到：

$$
i \le \sqrt n
$$

因此可以直接开一个数组，预处理：

$$
sum_2[i]
=
\sum_{j=1}^{i}
j \times \left\lfloor \frac{n}{j^2} \right\rfloor
$$

也就是第二部分的前缀和。

需要查询区间 $[l,r]$ 时：

$$
\sum_{i=l}^{r}
i \times \left\lfloor \frac{n}{i^2} \right\rfloor
=
sum_2[r]-sum_2[l-1]
$$

但是要注意，如果 $r > \sqrt n$，那么超过 $\sqrt n$ 的部分都是 $0$，所以可以把右端点限制在 $\sqrt n$ 以内。

也就是查询时使用：

$$
sum_2[\min(r,\sqrt n)] - sum_2[\min(l-1,\sqrt n)]
$$

## 6. 最终答案

根据最开始的拆式：

$$
\left\lfloor \frac{n}{i} \right\rfloor \bmod i
=
\left\lfloor \frac{n}{i} \right\rfloor
-
i \times \left\lfloor \frac{n}{i^2} \right\rfloor
$$

所以每个询问的答案是：

$$
ans
=
\left(sum_1(r)-sum_1(l-1)\right)
-
\left(sum_2(r)-sum_2(l-1)\right)
$$

最后对 $10^9+7$ 取模即可。

## 7. 思路总结

这道题主要分为两步：

1. 先利用取模定义，把原式拆成两部分：

$$
\left\lfloor \frac{n}{i} \right\rfloor \bmod i
=
\left\lfloor \frac{n}{i} \right\rfloor
-
i \times \left\lfloor \frac{n}{i^2} \right\rfloor
$$

2. 再分别处理两部分：

- 第一部分 $\displaystyle \sum \left\lfloor \frac{n}{i} \right\rfloor$ 用整除分块，并预处理每块的前缀和；
- 第二部分 $\displaystyle \sum i \times \left\lfloor \frac{n}{i^2} \right\rfloor$ 只需要处理到 $\sqrt n$，直接预处理前缀和即可。

这样每次询问时：

- 第一部分用二分找到所在块；
- 第二部分直接用前缀和；
- 最后两部分相减得到答案。





```cpp
#include <bits/stdc++.h>
using namespace std;

const long long MOD = 1e9 + 7;
struct Infoma{
    long long l;
    long long r;
    long long val;
    long long pre_sum;

    bool operator< (long long kk) const{
        return r < kk;
    }
};
vector <Infoma> blocks;
vector <long long> pre2;

long long sum_fir (long long x) {
    if (x <= 0)  return 0;
    auto pos = lower_bound (blocks.begin (), blocks.end (), x);

    long long real_pos = pos - blocks.begin ();

    long long res = 0;
    if (real_pos != 0) {
        res += blocks[real_pos - 1].pre_sum;
    }
    res = (res + (x - blocks[real_pos].l + 1) % MOD * (blocks[real_pos].val) % MOD) % MOD;
    return res;
}

long long sum_sec (long long x, long long sq) {
    if (x > sq)  return pre2[sq];
    return pre2[x];
}

int main ()
{
    ios::sync_with_stdio (false);
    cin.tie (0);

    long long n, q;
    cin >> n >> q;

    long long pre = 0; 
    for (long long l = 1; l <= n;) { // 预处理第一部分分块
        long long val = n / l;
        long long r = n / val;

        long long len = r - l  + 1;
        pre = (pre + ((val % MOD) * (len % MOD)) % MOD) % MOD;

        blocks.push_back ({l, r, val, pre});
        l = r + 1;
    }

    // 保证sq <= n
    long long sq = sqrt (n);
    while (sq * sq <= n)  sq ++;
    while (sq * sq > n) sq --;

    pre2.assign (sq + 1, 0);
    for (long long i = 1; i <= sq; i ++) {
        long long down = n / (i * i);
        pre2[i] = (pre2[i - 1] + i % MOD * (down) % MOD) % MOD;
    }

    while (q --) {
        long long l, r;
        cin >> l >> r;

        long long part1 = (sum_fir (r) - sum_fir (l - 1) + MOD) % MOD;
        long long part2 = (sum_sec (r, sq) - sum_sec (l - 1, sq) + MOD) % MOD;
        long long ans = (part1 - part2 + MOD) % MOD;

        cout << ans << '\n';
    }
    return 0;
}
```
