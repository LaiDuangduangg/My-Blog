### 题目描述

设 $f(A, B)$ 为 $A, A + 1, ..., B$ 的异或运算结果。求 $f(A, B)$ 的值。


### 约束条件

- 输入中的所有值均为整数。
- $0 \le A \le B \le 10^{12}$

### 输入

输入以标准格式给出：
`A`  `B`


### 输出

计算 $f(A, B)$ 并打印结果。


## Sample 1

**Input**

```text
2 4
```

**Output**

```text
5
```
* 2,3,4分别为010、011、100的二进制表示。 这些数的异或结果为101，用十进制表示为5


## Sample 2

**Input**

```text
123 456

```

**Output**

```text
435

```


## Sample 3

**Input**

```text
123456789012 123456789012

```

**Output**

```text
123456789012

```

## 题目解析：
* 在`异或前缀和的 1.0`讲过模板,不会的直接移步.

## 代码：
```cpp
#include <bits/stdc++.h>
using namespace std;

long long solve (long long a) {
    if (a == -1) return 0;
    if (a == 0)  return 0;
    else if (a % 4 == 1) return 1;
    else if (a % 4 == 2)  return a + 1;
    else if (a % 4 == 3) return 0;
    else if (a % 4 == 0) return a;
}

int main ()
{
    long long a, b;
    cin >> a >> b;

    long long ans1 = solve (b);
    long long ans2 = solve (a - 1);
    cout << (ans1 ^ ans2) << '\n';

    return 0;
}
```
