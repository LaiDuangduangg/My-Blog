## 缘由

假设有：

> $1, 2, 3$

如果题目说“任选一些数异或”，直接枚举子集，需要枚举：

> 空集  
> {1}  
> {2}  
> {3}  
> {1, 2}  
> {1, 3}  
> {2, 3}  
> {1, 2, 3}

一共 $2^3 = 8$ 种。

如果 $n = 5 \times 10^5$，显然不可能枚举 $2^n$ 个子集。

但是 XOR 有一个特殊性质：

> $1 \operatorname{xor} 2 = 3$

所以 $3$ 并没有提供新的信息，它只是 $1$ 和 $2$ 的组合。

所有子集异或结果其实存在大量重复。线性基就是用来删除这些“重复方向”的。


也就是说,线性基类似多项式的线性组合, 一条式子本身只需要选取其中一些线性无关的式子组合而成, 那些可以相互表示的部分可以直接去掉.
到具体情境中，即一些异或值可以用已有的值进行表示而无需引入新的值，

## 什么叫“线性无关”？

先看：

> $1 = 001$  
> $2 = 010$  
> $3 = 011$

因为：

$$
1 \operatorname{xor} 2 = 3
$$

所以有了 $1$ 和 $2$ 后，$3$ 就是重复信息。

我们说：

> $1$ 和 $2$ 是独立的；  
> $3$ 可以由它们组合出来，不独立。

在线性基中，一组数“线性无关”的意思是：

> 任意选择其中一些数进行 XOR，都不会得到 $0$。  
> 换句话说，没有一个数能由其他数 XOR 得到。

线性基会保留所有独立信息，删除所有重复信息。


## 存储
线性基实际上保存的是：

> **原数组能够通过子集 XOR 得到的所有值组成的集合。**

例如：

> $5 = 101$  
> $3 = 011$  
> $6 = 110$

因为：

> $6 = 5 \operatorname{xor} 3$

所以只保留 `5` 和 `3` 就够了。

这两个基向量可以产生：

> $0$  
> $5$  
> $3$  
> $5 \operatorname{xor} 3 = 6$

刚好是原数组所有不同的子集异或结果。

原来可能有很多个数，但最终只需要保留：

> `rank` 个独立向量

如果数最多只有 30 位，那么 `rank` 最多就是 30。

## 线性基表示法

我们用数组 `b[i]` 保存线性基。

其中：

> `b[i]` 表示一个最高位恰好是第 $i$ 位的数。

例如：

> `b[2] = 101`  
> `b[1] = 011`

这里：

- `101` 的最高位是第 $2$ 位；
- `011` 的最高位是第 $1$ 位；
- 不会有两个数放在同一个 `b[i]` 中。

这和高斯消元中的“主元”是同一个想法。

---

## 插入一个数

假设当前已有：

> `b[2] = 101`  
> `b[1] = 011`

现在插入：

> $x = 110$

从最高位开始看。

`110` 的最高位是第 $2$ 位，但 `b[2]` 已经有 `101`，于是用它消掉这一位：

$$
110 \operatorname{xor} 101 = 011
$$

现在 `011` 的最高位是第 $1$ 位，而 `b[1]` 已经有 `011`：

$$
011 \operatorname{xor} 011 = 000
$$

最后变成了 $0$。

这说明：

> $110$ 可以由已有线性基表示，  
> 它没有带来新的信息，不需要加入线性基。

# 题目描述

给定 $n$ 个整数（数字可能重复），求在这些数中选取任意个，使得它们的异或和最大。

## 输入格式

第一行一个数 $n$，表示元素个数。

接下来一行 $n$ 个数。

## 输出格式

输出一行一个整数，表示答案。

## 输入输出样例

### 输入 #1

```text
2
1 1
```

### 输出 #1

```text
1
```

### 输入 #2

```text
4
1 5 9 4
```

### 输出 #2

```text
13
```

## 说明/提示

$1 \le n \le 50,\ 0 \le S_i < 2^{50}$。

## 代码如下:
```cpp
#include <bits/stdc++.h>
using namespace std;

long long b[70];

void insert (long long x) {
    for (int i = 63; i >= 0; i --) {
        if (((x >> i) & 1) == 0)  continue;

        if (b[i] == 0) {
            b[i] = x;
            return;
        }
        x ^= b[i];
    }
}

long long max_xorval (long long start) {
    for (int i = 63; i >= 0; i --) {
        if (b[i] != 0 && (b[i] ^ start) > start) {
            start ^= b[i];
        }
    }
    return start;
}

int main ()
{
    ios::sync_with_stdio (false);
    cin.tie (0);

    long long n;
    cin >> n;

    for (int i = 0; i < n; i ++) {
        long long x;
        cin >> x;
        insert (x
        );
    }
    long long ans = max_xorval (0);
    cout << ans << '\n';
    return 0;
}
```
